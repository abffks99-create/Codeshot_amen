# 🛡️ CodeShot - LLM 기반 피싱 탐지 웹 서비스

Flask 기반 웹 애플리케이션으로,  
**Gemini API + RAG(검색 기반 생성)**를 활용하여  
URL 및 이미지의 피싱 여부를 분석하는 서비스입니다.

---

## 🚀 주요 기능

### 🔐 사용자 기능
- 회원가입 / 로그인 / 로그아웃
- 개인 분석 이력 관리
- 분석 결과 저장 및 조회

### 🌐 URL 분석 (크롤링 기반)
- 입력된 URL을 직접 크롤링
- 페이지 제목, 본문, 링크, 이미지 등 데이터 수집
- RAG + Gemini를 활용한 피싱 위험도 분석

### 🖼️ 이미지 분석
- 이미지 업로드 (피싱 문자, 화면 캡처 등)
- 이미지 내용을 기반으로 피싱 여부 판단
- JSON 구조로 상세 분석 결과 제공

### 🧠 AI 기반 RAG 분석
- 3가지 데이터 소스 활용:
  1. 피싱 URL 블랙리스트
  2. 과거 피싱 사례
  3. 금융감독원·KISA 공식 패턴
- 벡터DB(ChromaDB) 기반 유사도 검색
- Gemini 모델과 결합하여 최종 판단

### ⚡ 실시간 위협 탐지
- OpenPhish 피드 자동 업데이트
- 최신 피싱 URL 반영 (12시간 주기)
- 블랙리스트 기반 즉시 위험 판별

## 💡 RAG를 사용하는 이유
---기존 LLM은 단순 생성 기반이지만, 본 시스템은 실제 피싱 데이터(블랙리스트, 사례, 패턴)를 기반으로 판단하여 더 신뢰도 높은 분석을 제공합니다.


## 🏗️ 시스템 아키텍처

1. 사용자 입력 (URL / 이미지)
2. 블랙리스트 기반 1차 필터링
3. 웹 크롤링 및 데이터 수집
4. RAG 기반 유사 사례 검색
5. Gemini AI 최종 분석
6. 결과 반환 (위험도 및 설명)
